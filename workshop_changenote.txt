## v1.1.6 — Active Lazes + Projectile Tracking + Follow-up Fixes (March 22, 2026)

**What’s New:**

- Added **Active Lazes** so live laser designation points appear on the map for connected clients
- Added **Projectile Tracking** with predicted flight paths and ETA labels for major ordnance
- Improved fired-event targeting data using assigned targets, laser targets, and scanned laser objects
- Captured vehicle-fired heavy ordnance more reliably with dedicated vehicle `Fired` handlers

**Follow-up Improvements (after release):**

- Projectile direction now uses turret/fire vector data for artillery and vehicle weapons (instead of hull-only direction)
- End-to-end fired-event direction flow aligned across SQF, backend, and frontend
- CIV map/ORBAT filtering and color handling corrected
- Tree/object rendering orientation and visibility tuned for clearer map output

**BattlEye / DLL Note:**

- Current DLL-based sessions should still use `-noBE` for singleplayer/LAN until the 1.1.6 extension build is allowlisted
- Example: `arma3.exe -mod=@AthenaRemastered -noBE`
