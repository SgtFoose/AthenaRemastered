# Athena Remastered v1.1.6

This page highlights what shipped in v1.1.6.

## Nr.1 Active Lazes

There is a special moment in any operation when talk turns into precision. With Active Lazes, teams can now see exactly where laser designators are painting the map in real time. Instead of guessing where support should land, everyone shares the same truth at the same second.

A JTAC marks. A pilot confirms. A fire mission adjusts. The map becomes a living coordination board where intent is visible, not implied.

![Athena Remastered Active Lazes](../Images/Athena%20Remastered%20Active%20Lazes.png)

## Nr.2 Projectile Tracking

Projectile Tracking is all about turning chaos into confidence. When rounds and missiles are in flight, users can follow the shot path and understand what is happening before impact. This gives squads, vehicle crews, and command elements a better read on battlefield tempo.

The goal is not just visual flair. It is decision speed:

- Identify incoming or outgoing fire paths quickly.
- Understand likely target direction and timing at a glance.
- Keep team communication grounded in what the map is actually showing.

Together, Active Lazes and Projectile Tracking push Athena Remastered closer to a map that does not only show positions, but also intent and action.

![Athena Remastered Active Lazes and Projectile Tracking](../Images/Athena%20Remastered%20Active%20Lazes%20and%20Projectile%20Tracking.png)

## Nr.3 Post-release polish (March 22, 2026)

After release, we pushed a focused quality pass to improve reliability in live sessions:

- Projectile heading now prefers turret-based fire direction (`weaponDirection`) instead of vehicle hull direction for artillery and vehicle weapons.
- Fired-event data path is now end-to-end aligned (SQF -> Backend -> Frontend) for consistent trajectory rendering.
- CIV filtering and color behavior in ORBAT/map presentation were corrected.
- Tree/object map rendering was tuned for better orientation consistency and visibility at practical zoom levels.

## Release Note

v1.1.6 is now live with Active Lazes and Projectile Tracking, followed by stability and visualization improvements based on field testing.
