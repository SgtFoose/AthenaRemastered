[h1]Athena Remastered — Real-Time Tactical 2nd Screen Live Map[/h1]

A faithful remaster of Bus's original Athena — stream your Arma 3 mission to a second screen in real time. Works on any browser, tablet, or phone on your local network.


[hr][/hr]

[h1]Features[/h1]

[list]
[*] [b]Live Force and Faction Tracking[/b] — All units, vehicles, and groups rendered in real time on an interactive map
[*] [b]Full Terrain Rendering[/b] — Roads, forests, coastlines, contour lines, buildings, and named locations exported directly from the game engine
[*] [b]Multiple Map Styles[/b] — Switch between 2D Topographic, Ground Heatmap, and Pilot Heatmap views
[*] [b]NATO APP-6 Symbology[/b] — Group markers with proper echelon indicators
[*] [b]Vehicle Recognition[/b] — Distinct icons for tanks, APCs, helicopters, planes, boats, artillery, drones, and more
[*] [b]Role-Specific Unit Icons[/b] — Infantry, medic, officer, engineer, recon, MG, AT, and more
[*] [b]Real-Time Event Feed[/b] — Live kill feed and shot tracking
[*] [b]ORBAT Panel[/b] — Order of Battle view showing all groups organized by faction
[*] [b]Multi-Device Support[/b] — Connect multiple browsers, tablets, or phones to a single session
[*] [b]Server Admin Controls[/b] — Mission makers can toggle visibility of EAST/GUER/CIV sides via mission parameters
[/list]


[hr][/hr]

[h1]How It Works[/h1]

Athena Remastered runs a lightweight local server alongside Arma 3.

The project is transitioning from a C++ extension pipeline to a DLL-free pipeline:
[list]
[*] In-game SQF scripts collect mission/world/frame/event data on a schedule
[*] An external companion bridge service forwards telemetry to the local Athena backend
[*] Backend streams updates to browsers via WebSockets
[/list]

[code]
Arma 3 SQF → Bridge Service → Local Web Server → Your Browser
[/code]

No external servers. No cloud. Everything stays on your machine or local network.

[h1]Quick Start[/h1]

[olist]
[*] [b]Subscribe[/b] to this Workshop item
[*] [b]Open the server folder:[/b] [b]C:\Program Files (x86)\Steam\steamapps\common\Arma 3\!Workshop\@Athena Remastered\Server[/b]
[*] [b]Run the server:[/b] Start [b]AthenaRemastered.Server.exe[/b]
[*] [b]Open your browser:[/b] Go to [url=http://localhost:5000]http://localhost:5000[/url]
[*] [b]Launch Arma 3[/b] with the mod enabled
[*] [b]Start a mission[/b] — the map populates automatically!

[h1]Server Admin Notes[/h1]

[h2]Public Key (BIS Signature Verification)[/h2]

This mod ships with a signed PBO and a public key for server signature verification. To allow Athena Remastered on your server:

[olist]
[*] Navigate to the mod's [b]keys[/b] folder: [b]@AthenaRemastered\keys\[/b]
[*] Copy [b]AthenaRemasteredKey.bikey[/b] into your server's [b]Keys[/b] folder (next to your server executable)
[*] Ensure [b]verifySignatures = 2;[/b] is set in your server config — Athena's PBO will pass verification
[/olist]

The key file is also available in the mod's Workshop folder after subscribing.

[h2]Side Visibility Controls[/h2]

Mission makers can control what sides are visible to Athena through mission parameters in [b]description.ext[/b]:
[list]
[*] Show/hide EAST forces
[*] Show/hide Guerrilla forces
[*] Show/hide Civilian units
[/list]

These settings are pushed automatically from the mission to all connected browsers.
